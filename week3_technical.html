Looping

The While Loop



This particular loop executes code while the conditional statement is true and doesn't stop executing it until the conditional statement is false.



For example:

i = 0

while i < 3 do

puts "hi"

i = i + 1

end



This will run the code 3 times because i is less than 3 3 times since it is incrementing by 1. You can also make use of a begin statement and put the while and conditional statement after the end that is for the begin. This will do the same thing.

The Until Statement



This loop executes code while the conditional statement is false. It is a reverse of the while loop.



For example:

counter = 0

until 2 > counter do

puts "hi"

counter = counter + 1

end



This will perform the loop 2 times and therefore, print "hi" twice. You may also use the begin statement with the until loop. This is similar to the while loop.

The For Loop



This loop looks a little differently than it does in some other programming languages, but still works similar. We define a range of times to execute a specific block of code and it performs it those amount of times. It is possible to manipulate the variable and lengthen the loop.



Here is an example:

x for i in 0..3

puts "#{i} "

end



This code will print 0 1 2 3. This is really nifty if you know a specific amount of times to run the code.

The next four statements serve as support and help to assist the loop.



Break: Terminates the internal loop that it is in. It is really handly for ensuring that the program doesn't go into an infinite loop.

Next: Makes the program go to the next iteration of the most internal loop. It serves as an incrementor and is darn good at it.

Redo: This one does pretty much what you think it would. It restarts the iteration of the most internal loop. You just need to be careful that you don't put yourself in an infinite loop.

Retry: Serves as a rescue expression. If you are familiar with Java, retry is similar to try and catch blocks. It handles an error and restarts the code from the beginning of the begin body.
